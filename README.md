# BabyGuardian

Smart Baby Monitoring - A React website built from the Figma design.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Pages

- **Home** (`/`) - Hero, features, stats, CTA
- **About Us** (`/about`) - Company story, values, mission & vision
- **Our Solution** (`/solution`) - Core features, how it works
- **Blog** (`/blog`) - Articles, newsletter signup
- **Contacts** (`/contacts`) - Contact form
